# Terminal常用快捷键 #
| 组合键 | 作用 |
| --- | --- |
| Alt+Ctrl+T | 打开终端 |
| Shift+Ctrl+T | 以新标签页打开新终端 |
| Shift+Ctrl+N | 以新窗口打开新终端 |
| Shift+Ctrl+W | 关闭当前页终端 |
| Shift+Ctrl+Q | 关闭当前窗口所有终端 |
| Shift+Ctrl+C | 复制 |
| Shift+Ctrl+V | 粘贴 |
| Ctrl+__+__ | 放大终端字体 |
| Ctrl+__-__ | 缩小终端字体 |
| Ctrl+0 | 普通大小终端字体 |
| F11 | 全屏 |
| Ctrl+PgUp | 切换到上一个终端 |
| Ctrl+PgDn | 切换到下一个终端 |
| Shift+Ctrl+PgUp | 将终端向左移动 |
| Shift+Ctrl+PgDn | 将终端向右移动 |
