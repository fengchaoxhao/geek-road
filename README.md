## Win10常用快捷键 ##

- Windows+E  
打开文件资源管理器  
- Windows+左/右  
改变当前窗口的大小  
- Windows+上/下  
改变当前窗口的大小  
- Windows+Tab  
任务视图
- Alt+Tab  
切换窗口
- Windows+Ctrl+D  
创建新的虚拟桌面  
- Windows+Ctrl+F4  
关闭当前虚拟桌面  
- Windows+Ctrl+左/右  
左右切换虚拟桌面
- Windows+I  
打开Windows设置
- Windows+X  
打开高级用户功能
- Windows+P  
投影屏幕  
- Windows+L  
锁定屏幕  
- Windows+M  
最小化所有窗口  
- Windows+Shift+M  
还原最小化的窗口
